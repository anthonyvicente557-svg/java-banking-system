# 💰 Java Banking System

Sistema bancário simples desenvolvido em Java.

## 🚀 Funcionalidades
- Ver saldo
- Depositar dinheiro
- Sacar dinheiro
- Validação de valores
- Tratamento de erros

## 🛠️ Tecnologias
- Java
- IntelliJ IDEA

## 📚 Objetivo
Projeto criado para praticar lógica de programação e conceitos básicos de Java.
